# Flatiron Sinatra Portfolio Project 

Write your thoughts on the places you've visited during your travels 

# Installation 

1.Clone repo
2.CD into repo 
3.Run `bundle install`
4.Run `rake db:migrate`
    . Might already be created
5.Run Shotgun
6.Sign up and start your journal
